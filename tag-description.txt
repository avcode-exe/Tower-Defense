Tower Defense 1.5.0 Beta 1

Release notes:
- Healer troop is now available as a support unit that heals damaged allies and can be upgraded with TGT for more simultaneous heal targets.
- Players can manually heal Healer troops with gold, matching the behavior of other troop types.
- Healer troops no longer heal other support/healer troops, keeping healer-to-healer loops disabled.
- Save data validation and restore logic now clamp/validate Healer target counts and reject invalid HP values.
- Update detection now handles stable vs pre-release versions correctly, including pre-release users upgrading to newer stable releases.
- Added regression coverage for Healer behavior, update filtering, wave queue scaling, and persistence.
- README and project scripts were updated to reflect the current test suite and behavior.

Build notes:
- Windows x64 installer is generated by electron-builder.
- Auto-update metadata is generated automatically by electron-builder.
- Do not manually edit generated installer or latest update metadata files.
