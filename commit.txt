v1.2.0 prep: Troop healing system & UI layout fixes

Troop Healing:
- Players can heal troops by 10% of max HP per click
- Each heal costs 10% of the troop's base price (rounded up)
- Unlimited heals until troop reaches max HP
- Heal button always visible: green when affordable, grey when
  not enough gold, "MAX HP" greyed out when full
- Healing costs do NOT affect sell refund value

UI Layout:
- Moved upgrade buttons, heal button, and troop info panel up
  to eliminate overlaps between interactive elements
- Widened sell button to match shop panel width (SHOP_WIDTH - 16)
- Troop info panel shifted up 10px for clearance from upgrade buttons

Bug Fixes:
- Fixed troop selection jumping when other troops die (track by
  reference instead of index during compaction)
- Fixed monster double-attack on initial engagement (attackTimer
  now starts at full cooldown instead of 0)
- Fixed heal button disappearing when healCount reached cap
  (removed cap, button now always renders)
