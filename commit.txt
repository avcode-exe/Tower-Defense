feat: Milestone 2 — Healer Troop with lock-on healing, multi-target upgrade, and heal beam visual

Add the Healer as a new support-type troop that heals damaged allies instead of
attacking monsters. This closes Milestone 2 from the project roadmap.

Core behavior (src/troop.js):
- New Troop type 'support' with dedicated update branch that heals allies on
  attack-speed cadence instead of attacking monsters.
- pickHealTarget() locks onto the first damaged ally in range (by proximity),
  evicts healed/dead/out-of-range targets, and fills empty slots from closest
  candidates. Healing only stops or switches when the current target is fully
  healed.
- healTargets[] array tracks locked heal targets; healBeam property on healed
  troops tracks the beam source and fade timer.
- getHealAmount() returns cached damage stat as the per-tick heal value.
- New healTargetLevel stat (default 1) controls max simultaneous heal targets.
- Repurposed 'slow' upgrade slot for support troops as 'TGT' (target count).

Config (src/config.js):
- Healer spec: id='healer', type='support', cost=150, damage=8, range=3,
  attackSpeed=1.5, color='#2ecc71', hp=40.
- Stats string loop handles support type with 'heal' instead of 'dmg'.
- Updated description to mention multi-target healing.

Rendering (src/rendering/gameRenderer.js):
- Heal beam: solid semi-transparent yellow line (#f1c40f) drawn from healer to
  each recently-healed ally, fading out over 0.6s via healBeam.timer.
- Support troop center dot rendered in white to distinguish from melee (gold)
  and ranged (grey).

Particles (src/particles.js):
- healBurst green particle config for heal visual effects.

Shop UI (src/ui/shop.js):
- Type badge shows 'SUPRT' with green color for support troops.
- Stat panel shows HEAL/SPD/RNG/TGT instead of DMG/SPD/RNG/SLW for support.
- Upgrade buttons: HEAL (green), RNG, SPD, TGT (orange), HP for support.
- Heal button hidden for support troops (they auto-heal allies).

Placement (src/ui/placement.js):
- Green range circle in placement ghost for support troops.
