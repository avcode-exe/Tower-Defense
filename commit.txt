dev: Replace visible DEV button with hidden triple-click activation

Dev mode is now activated by triple-clicking the gold coin icon
in the HUD (3 clicks within 800ms). The visible DEV button and
F2 keyboard shortcut have been removed, making dev mode access
less obvious to casual players.

Three clicks are required: the hitbox is centered on the yellow
gold coin at the top-left of the screen (x:10-36, y:14-42).
Confirmation dialog still fires before toggling.

UI Layout:
- Widened dev-mode monster spawn panel from 180px to 220px
  to eliminate overlap between monster names (e.g. "Grunt x8")
  and the +/- control buttons
- Widened confirmation dialog from 340px to 380px for better
  text readability and button spacing
