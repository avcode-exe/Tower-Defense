Release 1.5.0 beta 1

Ship the 1.5.0 beta 1 release with the completed Healer support pass, save/update hardening, and GitHub release automation.

Highlights:
- Add Healer troop support with lock-on healing, multi-target upgrades, heal beams, and target-count persistence.
- Allow players to heal Healer troops with gold like other troops.
- Prevent Healer troops from healing other support/healer troops.
- Harden save restore validation for Healer target count and invalid HP values.
- Improve update filtering so pre-release users can upgrade to newer stable releases.
- Add regression tests for update filtering, wave queue scaling, persistence, and Healer behavior.
- Update README docs and package scripts for the expanded test suite.
