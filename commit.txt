Release 1.5.1-beta.2: Healer rework with monster damage

Healer troop rework:
- Reduced heal radius from 3 to 2 blocks
- Healer now deals 2 damage to every monster in its healing range
- Hitspeed reduced to 0.5s (from 1.5s) for faster heal/damage cadence
- Added monsterDamage field to healer spec and stats string display

Code changes:
- src/config.js: healer range 3→2, attackSpeed 1.5→0.5, added monsterDamage: 2
- src/troop.js: added getHealRangePxSq(), getMonsterDamage(), damageMonstersInHealRange()
- src/troop.js: support update no longer skips monster damage when no heal targets
- src/troop.js: uses game._monsterTileIndex when available for efficient AoE

Version bump:
- 1.5.1-beta.1 → 1.5.1-beta.2 across package.json, package-lock.json, updateManager.js, gamePersistence.js
