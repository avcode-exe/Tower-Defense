Release 1.5.1 beta 1: Necromancer monster

This beta release captures the Necromancer milestone 3 work under the 1.5.1-beta.1 pre-release version. It is intended for beta validation and follow-up fixes, not as a stable public release.

Necromancer changes:
- Adds monster key Y with noSplit behavior.
- Adds Runner noSplit behavior.
- Implements Necromancer revive behavior: up to 5 monsters within 2 blocks, skips other Necromancers and monsters that have reached the end, restores 30% HP, and adds blue glow/particles.
- Adds normal wave support plus dev UI/dev spawn queue support.
- Adds tests for config, wave/dev queue, no-split behavior, and revive targeting/range/limit/locks/glow.

Version metadata:
- Updated app and package metadata to 1.5.1-beta.1.
- Updated default update settings, save serialization, and release versioning expectations to 1.5.1-beta.1.

Validation passed:
- npm test
- npm run lint
- npm test -- tests/versioning.test.js

Formatting note:
- npm run format:check still reports warnings for src/gamePersistence.js, src/ui/preview.js, and src/updateManager.js. These formatting warnings were present before this beta version bump; format:check was not used as a gate for this beta commit.
