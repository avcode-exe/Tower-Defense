v1.4.0-beta.4 — Game class refactor, reset button hitbox fix, mute button

Refactoring (Game class decomposition):
- Extracted GameRuntimeController (js/gameRuntime.js) — owns Web Worker
  lifecycle, pause/resume render loop, resize subscription, and centralised
  state transitions (startWave, pauseGame, resumeGame, togglePause,
  applyDefeat, stopLoop, startLoop)
- Extracted SaveSerializer, GameWorldFactory, GameSnapshotRestorer
  (js/gamePersistence.js) — single touch-point for save/restore/restart
  logic; constructor and restart both reuse GameWorldFactory.createFresh
- Game.start(), restart(), restore(), getSaveData() now one-liners that
  delegate to the extracted helpers; all duplicated worker/state/RAF
  management removed from Game class
- pause/resume branches in onMouseDown, onKeyDown, and step() defeat
  handler collapsed from 4-5 call sites to single method calls
- index.html loads new modules before game.js; main.js dev-start-btn
  guarded with state check

Bug fix:
- Reset button hitbox (LAYOUT.HUD.RESET_BTN) updated from w:36 to w:50
  to match the visual width of the "Reset" label (was sized for old "RST")

New feature:
- Added mute/unmute button in top HUD, right of Reset button
- AUDIO.toggleMute() saves and restores previous volume level
- Button shows music note when active, red X when muted
- Cursor changes to pointer on hover, click toggles mute

Version bump:
- package.json, package-lock.json, gamePersistence.js save version
  bumped to 1.4.0-beta.4

Files changed:
- config.js: Added MUTE_BTN layout constant, fixed RESET_BTN width
- audio.js: Added muted getter, toggleMute() with volume save/restore
- game.js: Delegated lifecycle/persistence to runtime/persistence modules,
  added mute button hit test and click handler
- gamePersistence.js: New file — SaveSerializer, GameWorldFactory,
  GameSnapshotRestorer
- gameRuntime.js: New file — GameRuntimeController class
- index.html: Added script tags for gameRuntime.js, gamePersistence.js
- main.js: Guarded dev-start-btn with state check
- ui.js: Drew mute button in HUD, added it to drawHUD
- package.json: Version bumped to 1.4.0-beta.4
- package-lock.json: Version bumped to 1.4.0-beta.4
